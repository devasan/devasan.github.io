JSONModel的学习使用

发表于 2014-11-06   |   分类于 学无止境   |  

正在学习中，先占个坑，晚上回去争取填上。

—————————————————————————-—————————————————————-

回来填坑了。

今天还是在学习model的问题，想要学习一下JSONModel。

之前写model是这样的，然后拿到response可能还需要再各种objectForkey:

+ (UserRole )userRoleFromDict:(NSDictionary)dict
{
    UserRole *user = [[UserRole alloc]init];
    user.idx           = [dict objectForKey:@”idx”];
    user.username      = [dict objectForKey:@”username”];
    …..
    return user;
}

  • (NSDictionary )dictFromUserRole:(UserRole )userRole
    {
    NSMutableDictionary *dict = [[NSMutableDictionary alloc]init];
    [dict customSetValue:userRole.idx forKey:@”idx”];
    [dict customSetValue:userRole.username forKey:@”username”];
    ……

    return dict;
    }
    如果遇到几十个上百个参数，简直是不能忍。我这个暴脾气，重复这么多，每次写的都是火大。

好了，直接上干货。没有记错的话，TC也是用的runtime-class_copyPropertyList这套机制，当时很是喜欢，只要声明一下就可以了。可是代码巨多，看到哪都是迷迷糊糊的，看的是云里雾里的。

今天中午就照着开始写，然后遇到了一个问题，当然了，我这个问题现在看来很是傻逼。我去群里吼了一下，事实证明，QQ群是不适合讨论技术问题的，用来闲聊还挺不错的。Stackoverflow才是真正该去解决问题的地方。PS其实这个问题是QG给我道破的，把responseObject一直当json取了，就没注意已经是dictionary。回来的路上还在想，自己有过几次类似这样头脑短路了，把很简单的类型搞错了，然后弄了一大通。由此可见，适时的向别人请教是绝对必要的，QG果然是老江湖，几次遇到问题都让他给KO了。

#import “JSONModel.h”

@protocol ASJsonDiscoverData
@end
@interface ASJsonDiscoverData : JSONModel

@property (nonatomic,copy) NSString idx;
@property (nonatomic,copy) NSString logo;
@property (nonatomic,copy) NSString cover;
@property (nonatomic,copy) NSString title;
@property (nonatomic,copy) NSString intro;
@property (nonatomic,copy) NSString status;

@property (nonatomic,copy) NSString dateline;

@end

//可省略
@interface ASJsonDiscoverDataArray : JSONModel
@property (nonatomic,strong) NSArray <ASJsonDiscoverData>*data;
@end

@interface ASJsonDiscoverModel : JSONModel

@property (nonatomic,assign) int ret;
