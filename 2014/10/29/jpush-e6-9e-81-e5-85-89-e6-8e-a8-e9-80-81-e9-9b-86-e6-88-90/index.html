Jpush极光推送集成

发表于 2014-10-29   |   分类于 拾人牙慧   |  

整个集成过程比较简单，从开始集成到测试接收到推送一小时左右，过程如下：

拿到公司账号的时候，已经完成注册、添加应用资料等一系列操作，并且APNs证书文件(iOS)：开发环境: 已验证。生成证书步骤在这

接着下载官方SDK，将lib文件夹下的 APService.h和libPushSDK.a添加到项目。

添加一个名为“PushConfig.plist”Plist文件，有“CHANNEL”、“APP_KEY”、“APS_FOR_PRODUCTION”这三个字段，分别填入对应的键值内容。

接着在AppDelegate里添加代码

- (BOOL)application:(UIApplication )application didFinishLaunchingWithOptions:(NSDictionary )launchOptions
{
//极光推送相关

#if IPHONE_OS_VERSION_MAX_ALLOWED > IPHONE_7_1
if (systemGreaterthan_ios8) {
[APService registerForRemoteNotificationTypes:UIUserNotificationTypeAlert | UIUserNotificationTypeBadge |UIUserNotificationTypeSound categories:nil];
}else{
[APService registerForRemoteNotificationTypes:UIUserNotificationTypeAlert | UIUserNotificationTypeBadge |UIUserNotificationTypeSound categories:nil];
}

#else
[APService registerForRemoteNotificationTypes:UIUserNotificationTypeAlert | UIUserNotificationTypeBadge |UIUserNotificationTypeSound categories:nil];

#endif
[APService setupWithOption:launchOptions];

[self obeservJpushNotifications];

}

  • (void)obeservJpushNotifications
    {
    NSNotificationCenter *defaultCenter = [NSNotificationCenter defaultCenter];
    [defaultCenter addObserver:self

    selector:@selector(networkDidSetup:)
    name:kJPFNetworkDidSetupNotification
  object:nil];

    [defaultCenter addObserver:self

    selector:@selector(networkDidClose:)
    name:kJPFNetworkDidCloseNotification
  object:nil];

    [defaultCenter addObserver:self

    selector:@selector(networkDidRegister:)
    name:kJPFNetworkDidRegisterNotification
  object:nil];

    [defaultCenter addObserver:self

    selector:@selector(networkDidLogin:)
    name:kJPFNetworkDidLoginNotification
  object:nil];

    [defaultCenter addObserver:self

    selector:@selector(networkDidReceiveMessage:)
    name:kJPFNetworkDidReceiveMessageNotification
  object:nil];

    [defaultCenter addObserver:self

    selector:@selector(serviceError:)
    name:kJPFServiceErrorNotification
  object:nil];

    }

  • (void)networkDidSetup:(NSNotification *)notification {

    //NSLog(@”已连接”);
    }

  • (void)networkDidClose:(NSNotification *)notification {

    // NSLog(@”未连接。。。”);
    }

  • (void)networkDidRegister:(NSNotification *)notification {

    // NSLog(@”已注册”);
    }

  • (void)networkDidLogin:(NSNotification *)notification {

    //NSLog(@”已登录”);
    }

  • (void)serviceError:(NSNotification )notification {
    //错误信息
    NSDictionary     userInfo = [notification userInfo];
    NSString *error = [userInfo valueForKey:@”error”];
    NSLog(@”%@”, error);
    }
     

照着文档按部就班添加代码，这样就已经配置完了。（未对IOS 8的UIUserNotificationCategory操作）

进入账号网页测试推送，收到消息，但是推送的统计数据好像不是太准。在工程里加入音频文件，在推送消息时可自定义推送消息的提示音。

目前只是初步接收到推送，未做进一步处理。
