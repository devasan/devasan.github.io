iOS10适配

发表于 2016-12-29   |  

iOS10发布已经挺久了，这里记录一下本次APP发布过程中遇到的一些比较重要的适配事项，或者说是必须要适配的事项，还有一些API的更新，因为开发过程中要兼容较低的版本，就不罗列出来了。

1. Privacy-Sensitive Data

打包上传到iTunes Connect过程中遇到的一次错误提示

This app attempts to access privacy-sensitive data without a usage description. The app’s Info.plist must contain an NSMicrophoneUsageDescription key with a string value explaining to the user how the app uses this data.

解决的方法很简单，在Target-Info添加Privacy - Microphone Usage Description说明即可。其他需要访问隐私数据的（比如：Privacy - Camera Usage Description、Privacy - Photo Library Usage Description、Privacy - Contacts Usage Description还有位置、蓝牙、日历、健康、提示事项等就不一一列举了）都必须在info.plist里添加相关属性。

2. ATS

本来WWDC中宣布2017年1月1日起所有开发应用必须升级成HTTPS(特殊情况可以和Apple沟通),现在推迟了截止时间，下面是Apple给出的新闻。

Supporting App Transport Security
December 21, 2016

App Transport Security (ATS), introduced in iOS 9 and OS X v10.11,
improves user security and privacy by requiring apps to use secure
network connections over HTTPS. At WWDC 2016 we announced that apps
submitted to the App Store will be required to support ATS at the end
of the year. To give you additional time to prepare, this deadline has
been extended and we will provide another update when a new deadline
is confirmed. Learn more about ATS.

在iOS9的时候，就默认禁止非HTTPS的网络访问。需要我们在info.plist里添加App Transport Security Settings字典，将Allow Arbitrary Loads设置成YES，在Exception Domains添加域名访问白名单。

3. UserNotifications

#import <UserNotifications/UserNotifications.h>
现在的通知有了状态的回调，推送发出后，会得到一个response以及error，可以根据回调返回的状态处理不同的状况。
